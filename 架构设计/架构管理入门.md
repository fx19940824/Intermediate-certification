# 架构
- TOGAF-企业架构分类
    - 企业架构（EA）
        - 业务架构
            - 业务组件
            - 业务流程
            - 组件分布
            - 组织机构
            - 绩效考核
        - IT架构
            - 数据架构：数据在系统中的流转、管理和存储
            - 技术架构：系统、技术组件的实现和部署策略
            - 应用架构：IT系统功能和技术实现内容
- 应用架构面向业务，根据业务确认应用系统的功能边界和关联关系
- 数据架构关注数据在系统中的流转、管理和存储
- 技术架构关注系统、技术组件的实现和部署策略
